SDL2-Crystal
============
Using https://github.com/manastech/crystal/tree/master/libs/sdl as a base, and updating it to SDL2

## Prerequisites
* libsdl2
* libsdl2-gfx
* libsdl2-ttf

`sudo apt-get install libsdl2-dev libsdl2-gfx-dev libsdl2-ttf-dev`

## How to Build
```
cd src/
crystal deps
crystal main.cr
```
